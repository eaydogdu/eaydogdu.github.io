For basic syntax of R, I point you to Learn R in Y Minutes. In the following I show basic work flows for data analysis.

I gradually switched to the tidyverse ecosystem, while using data.table in case I need faster manipulation.

Basic Usage

# load package
load(tidyverse)

# learn about a dataset
glimpse(dat)

# basic manipulation
dat %>%
    filter(!is.na(x)) %>% # filter observations
    mutate(y = x^2) %>%   # generate new variable
    select(-x) %>%        # delete variable
    order(y) %>%          # order by variable
    rename(z = y) %>%     # rename variables

Advanced Usage

Parallel

We can use the built-in parallel package. General mechanism includes the following

# initiate the cluster
ncore <- 6
cl <- makeCluster(ncore)

# copy the environment 
clusterExport(cl, varlist = ..., envir = environment())

# to load any packages, you can include the
#   library()
# argument in the function

# run the code
parLapply(cl, input, fun)

Tips

  1. Better regression format
    One way to manage control variables in R is to use the paste function.
control <- c("var1", "var2", ...)
control <- paste(control, collapse = "+")
reg1 <- lm(as.formula(paste("dep.var ~", control, sep = "")), 
           date = Data.Frame)
  1. Be careful about read converting strings to factors
