"Swiss Knife" for Paper Writing

Publish Date: Nov 4, 2016    Last Update: Jun 26, 2018

Table of Contents

  • First best
  • The Swiss Knife

This post documents my workflow for writing an academic article. Some useful tools help to make the process much more efficient.

First best

In my case, the First Best case will fulfill the following needs1

  1. Project management
    • todo list
    • manage issues/comments/ideas
    • version control
  2. Text editing environment that features
    • beautiful interface
    • powerful grammar check
    • easy collaboration
    • flexible format (conversion to pdf, html)
  3. Bibliography tool to manage all the papers

The Swiss Knife

I use a combination of tools to satisfy my needs. The ingredient list contains:

  1. Paperpile - Bibliography management tool
  2. Github - Project management and issue management
  3. Vim + Markdown - Writing environment
  4. Pandoc - Conversion tool

So this is how it works. For each project, I will create a Github repository for code and manuscript: code, data, writeup, etc. Github provides two great features that are especially suitable for academic writing: project and issue. I use project to manage ToDos, jot down quick ideas and resources. For issues that I need to solve, I will open an issue specifically for the question. Git is also designed for collaboration and version control. Overall, a Github repository forms a great foundation.

I write papers in Vim. For the source file, I use the markdown format, which features simple syntax and high readability. Vim has many tools specifically for better markdown support, so choose the one that fits you demand.

I use Paperpile as my bibliography management tools. Paperpile has several very neat features. First, it has a PDF reader that syncs all annotation in the cloud. Second, it syncs the PDF file with google drive, which means I can read the PDFs on the go with any other client that supports Google Drive. For a specific project, I will use a tag to mark all the cited papers. Once the paper is finished, I will just export all the bibliography information with that tag.

Hope this little post helps! Happy writing papers!
